pub struct Stack<T> {
    pub focus: T,
    pub up: Vec<T>,
    pub down: Vec<T>,
}

Handles focus tracking on a workspace. focus keeps track of the focused window's id and up and down are the windows above or below the focus stack respectively.

Immutable

Note that this Stack implementation is immutable and that each operation that would modify it, instead returns a new copy of the Stack with the modified state.
